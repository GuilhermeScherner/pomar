# Backend

Acesse a pasta do backend:

    cd backend/

## Como buildar

    docker-compose build

## Como iniciar

    docker-compose up

e depois acesse:

    http://localhost:8000

## Documentação

    swagger - http://localhost:8000/docs
    redoc - http://localhost:8000/redoc

## Pgadmin

    http://localhost:5050


# Frontend

Acesse a pasta do frontend:

    cd frontend/


## Como instalar

    yarn install

## Como iniciar
    
    yarn start
    
e depois acesse:
    
    http://localhost:3000
